# Formalization -- VERSEIM-REU 2025

This is the repository for the *formalization project* of the [2025
VERSEIM-REU](https://sites.tufts.edu/verseimreu/).

- The repository contains [`Lean` code](/VERSEIM2025/) produced
  during the program.
